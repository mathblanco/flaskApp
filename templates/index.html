Bem-vindo à Aplicação de Análise de Sentimento!

Esta aplicação pythonFlask utiliza o conjunto de dados um modelo de machine learning treinado para prever o sentimento (positivo ou negativo) de críticas de filmes. O modelo foi desenvolvido utilizando o algoritmo de Naive Bayes com vetorização de texto. O treinamento utilizou o IMDB Dataset, um conjunto de dados em inglês que contém avaliações de filmes e seus sentimentos associados
